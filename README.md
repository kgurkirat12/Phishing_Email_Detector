# 🛡️ Phishing Email Detector

## 📌 Project Description

The Phishing Email Detector is a web-based cybersecurity application developed using Python and Flask. It helps users identify potentially malicious phishing emails by analyzing the email content and calculating a risk score. The application classifies emails into High Risk, Medium Risk, or Low Risk categories based on predefined phishing indicators.

The project also stores analysis results in a SQLite database, allowing users to view previous analyses through a history page. An interactive dashboard built with Chart.js displays email statistics and risk distribution, providing an easy way to monitor overall email security.

---

## ✨ Features

- 📧 Analyze email content for phishing indicators
- 🎯 Calculate a phishing risk score (0–100)
- 🚨 Classify emails as High, Medium, or Low Risk
- 📊 Interactive dashboard with charts
- 📜 View analysis history
- 💾 Store results using SQLite database
- 📱 Responsive user interface using Bootstrap
- ⚡ Fast and lightweight Flask application

## 🚀 How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/kgurkirat12/Phishing_Email_Detector.git
```

2. Navigate to the project folder

```bash
cd Phishing_Email_Detector
```

3. Install the required packages

```bash
pip install flask
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
